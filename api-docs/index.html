📡 端点列表

GET /health

服务健康检查

示例响应
{"status": "ok", "time": "2026-05-26T10:00:00"}
GET /weather/current

广州实时天气

示例响应
{"current": {"temp": 30, "humidity": 60, "weather": "多云", "need_umbrella": false}}
GET /weather/history?days=7

天气历史数据

参数: ?days=7 (默认7天, 最大365)
示例响应
{"count": 24, "history": [{"record_date": "2026-05-26", "temp": 30, ...}]}
GET /weather/openmeteo?hours=0&limit=10

Open-Meteo 逐小时预报数据（hours=0 查全部）

参数: ?hours=24&limit=100 (hours=0 不限时间)
示例响应
{"count": 10, "records": [{"datetime": "2026-02-18 20:00", "temperature": 15.6, ...}]}
GET /huinan/states

回南天最新状态

示例响应
{"state": {"state": "unknown", "level": "none", "confidence": 0.0, "timestamp": "..."}}
GET /huinan/records?action=latest&limit=100

回南天记录查询（latest / range / stats）

参数: ?action=latest&limit=100
?action=stats
?action=range&start=...&end=...
示例响应
{"success": true, "total": 10, "records": [{"timestamp": "...", "temp": 26.5, "humidity": 85, ...}]}
GET /aqi/current

最新空气质量（AQI / PM2.5 / PM10 / NO2 / SO2 / CO / O3 / UV）

示例响应
{"current": {"aqi": 26, "pm25": 18.0, "pm10": 35.0, "level": "优", "record_time": "..."}}
GET /aqi/history?limit=100

空气质量历史

参数: ?limit=100 (最大1000)
示例响应
{"count": 5, "records": [{"aqi": 26, "pm25": 18.0, "record_time": "..."}]}
GET /system/load?limit=1

系统负载（CPU / 内存 / 磁盘 / 网络）

参数: ?limit=1 (返回条数, 最大1000)
示例响应
{"count": 1, "records": [{"cpu_percent": 5.2, "memory_percent": 43.1, "load1": 0.05, "disk_percent": 15.0, "record_time": "..."}]}
GET /website/checks?limit=10

网站可用性监控

参数: ?limit=100&site=xxx (按站点筛选)
示例响应
{"count": 5, "records": [{"site_name": "kisara.art", "status_code": 200, "response_time_ms": 230, "is_up": 1}]}
GET /ip/records?limit=1

IP 一致性检查记录

参数: ?limit=100 (最大1000)
示例响应
{"count": 1, "records": [{"hostname": "VM-45-5-ubuntu", "ifconfig_me": "82.157.11.118", "is_consistent": 1, "timestamp": "..."}]}

所有数据汇总（天气 + OpenMeteo + 回南天 + AQI + 系统负载 + 网站监控 + IP）

示例响应
{"server": {"api_time": "..."}, "weather": {...}, "openmeteo": {...}, "huinan": {...}, "aqi": {...}, "systemLoad": {...}, "websiteChecks": [...], "ipRecord": {...}}
