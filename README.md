## 📊 可视化服务

### 系统负载监控

实时监控系统性能指标：
- **CPU 使用率**：实时百分比和负载趋势
- **内存使用**：使用量和百分比  
- **磁盘使用**：存储空间监控
- **系统负载**：1分钟/5分钟/15分钟平均负载

**访问**: https://monitor.kisara.info/system-load-visualization.html

**技术特点**:
- 科幻风格界面（霓虹渐变、扫描线效果）
- Chart.js 实时图表
- 每15分钟自动采集（一天56个数据点）
- 自动避开 cron 任务高峰期
